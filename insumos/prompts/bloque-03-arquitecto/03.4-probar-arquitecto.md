# Prompt 3.4 — Probar el arquitecto sobre la spec

**Bloque 3 · Agente arquitecto**

```
CONTEXTO:
Voy a invocar al agente `arquitecto` recién creado sobre la spec del
proyecto (versión actualizada después de cerrar los huecos bloqueadores
detectados en el bloque 1).

OBJETIVO:
Que el agente proponga las 3 decisiones arquitectónicas mínimas del
proyecto: modelo de datos, estrategia de autenticación, frontera
cliente/servidor.

RESTRICCIONES:
(las que ya están en el system prompt del agente)

CRITERIO DE ÉXITO:
Para cada decisión: 2-3 alternativas con trade-offs reales. Cierre
con "¿cuál eliges?". Nada de decisiones tomadas por el agente.
```
