# Prompt 7.2 — Construir el agente reviewer

**Bloque 7 · Agente reviewer**

```
CONTEXTO:
Spec del reviewer finalizada.

OBJETIVO:
Materializar `.claude/agents/reviewer.md`.

RESTRICCIONES:
- YAML válido.
- System prompt fuerza al agente a:
  1. Leer AGENTS.md, spec.md, ADRs y luego el diff o commit pedido.
  2. Reportar problemas en 4 secciones (los 4 ejes de la spec).
  3. Marcar prioridad de cada problema: bloqueante / advertencia / nota.
  4. No reescribir código. Solo señalar.
- En español.

CRITERIO DE ÉXITO:
Agente creado. Funcional sobre un commit de prueba.
```
