# Prompt 7.3 — Smoke-test del reviewer sobre un commit real

**Bloque 7 · Agente reviewer**

```
CONTEXTO:
El agente reviewer ya existe en .claude/agents/reviewer.md. Quiero
probarlo sobre un commit real del repo (por ejemplo el de AGENTS.md)
para confirmar que funciona.

OBJETIVO:
Invocar al reviewer sobre el último commit y revisar que entrega un
reporte en sus 4 ejes con prioridad por problema.

RESTRICCIONES:
- El reviewer solo reporta, no arregla nada.
- El reporte debe cubrir los 4 ejes (AGENTS.md, commit atómico,
  alineación con spec/ADRs, rastros de debug) y marcar prioridad
  (bloqueante / advertencia / nota) en cada hallazgo.

CRITERIO DE ÉXITO:
Recibo un reporte estructurado en 4 ejes con prioridades. Si el agente
divaga o intenta arreglar código, ajusto su system prompt.
```

> Solo lectura: smoke-test, no busca bugs reales sino confirmar que el agente hace lo que su spec promete.

**Rama y commit:** `feat/agente-reviewer` desde `develop` → `feat: agente reviewer` → merge a `develop`.
