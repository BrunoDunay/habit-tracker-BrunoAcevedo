# Prompt 5.2 — Construir el archivo del agente diseñador

**Bloque 5 · Agente diseñador**

```
CONTEXTO:
Spec del agente `diseñador` finalizada después de iteración manual.

OBJETIVO:
Materializar el archivo `.claude/agents/diseñador.md`.

RESTRICCIONES:
- Encabezado YAML válido con name, description, tools (si aplica).
- El system prompt fuerza al agente a:
  1. Leer spec.md y AGENTS.md antes de proponer.
  2. Proponer 1 paleta de colores (3-5 colores con justificación
     funcional: primario, fondo, texto, error, éxito).
  3. Proponer 1 stack tipográfico (system font stack o Google Font,
     máximo 2 familias).
  4. Proponer escala de espaciado consistente basada en Tailwind.
  5. Listar componentes UI necesarios derivados de la spec.
  6. Listar páginas necesarias derivadas de la spec.
  7. Respetar restricciones de AGENTS.md.
- En español.

CRITERIO DE ÉXITO:
Agente creado y al invocarlo produce un documento de sistema visual
listo para guardar en `docs/diseño.md`.
```
