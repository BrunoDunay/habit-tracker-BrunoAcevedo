# Habit Tracker

Proyecto del Diplomado Vibe Coding Profesional.

## Descripción

Habit Tracker es una aplicación web para el seguimiento de hábitos personales. Permite a los usuarios crear hábitos, registrar check-ins diarios y visualizar su progreso mediante rachas y un calendario de cumplimiento.

El objetivo del proyecto es ofrecer una herramienta simple y estructurada para mantener consistencia en hábitos personales sin elementos sociales ni gamificación.
